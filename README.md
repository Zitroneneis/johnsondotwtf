# Jay Johnson - Personal Contact Page

This is my personal contact page hosted at [johnson.wtf](https://johnson.wtf) - a neobrutalist-styled landing page showcasing my work as a Senior Technical Product Manager.

## About This Page

This single-page application serves as my professional contact page and portfolio introduction. Built with modern web technologies and a bold neobrutalist design aesthetic, it features:

- **Rotating Cat Photos**: Cutie-patootie pictures of my cat Iris, who is the official PM Assistant of the Year 2026 🏆
- **Professional Introduction**: Overview of my 15+ years of experience in product management
- **Interactive Contact Modal**: Easy-to-use contact interface with one-click email copying
- **Responsive Design**: Optimized for all device sizes

## Technologies

- HTML5 & CSS3
- Tailwind CSS for styling
- Vanilla JavaScript for interactivity
- Google Fonts (Manrope) and Material Symbols
- Neobrutalist design principles (bold borders, shadows, and vibrant colors)

## Features

### Dynamic Content
- Rotating headshots of Jay looking quite dashing
- Rotating photos of Iris the cat (because every PM needs an assistant)
- Skill tag system
- Interactive contact modal with clipboard functionality

### Design Elements
- Neobrutalist shadows and borders
- Vibrant lemon yellow accent color (#e0ff1e)
- Bold typography and clear visual hierarchy
- Smooth animations and transitions
